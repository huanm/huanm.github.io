一、基于本机镜像

1.在root目录下将系统打包

tar --numeric-owner --exclude=/proc --exclude=/sys -cvf centos7.tar /

2.将制作的centos导入并命名为centos7

  1. 本地导入：
    cat centos7.tar | docker import - centos7
  2. 远程导入:
    docker import http://example.com/exampleimage.tgz

二、通过febootstrap工具

1.下载安装febootstrap

2.制作基础镜像

febootstrap -i bash -i wget -i yum -i iputils -i iproute centos6 centos6-image https://mirrors.aliyun.com/centos/6/os/x86_64/

(-i 安装package， centos6 操作系统版本，centos6-image安装目录，最后是源地址)

3.导入镜像

cd centos6-image && tar -c .|docker import - centos:6
